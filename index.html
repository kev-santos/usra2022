Assignment 1 feedback: *Note this is not comprehensive! Check the posted solutions as well. (updated June 7)

  • Write your full name as it appears on your Quercus account.
  • Don't write "stacks of equations". When writing a calculation, try to write a justification for each line.
  • If proving a fact about Riemann sums, begin by introducing a Riemann partition ("Let {x_i} be a Riemann partition...") so that x_i is defined.
  • To claim something about an entire sum, you need to show that it is true for each of the terms in the sum.
  • If you take out a term like "1/n" from a sum, make sure you indicate that 1/n is "constant with respect to the index".
  • To justify that a set is a partition of an interval [a,b] you need two things: (1) each point is an element of [a,b] AND (2) identify the delta x value.
  • A picture is not enough for a proof. Use a combination of calculations and words to write a detailed argument.
